# Book Store Management System (C++)

## Overview

The Book Store Management System is a console-based application developed in C++ as an academic project. It demonstrates the practical implementation of Object-Oriented Programming (OOP) concepts and Data Structures for managing bookstore operations. The system allows users to maintain book records, purchase books, search and update book information, sort records, and calculate the total inventory value through a simple menu-driven interface.

## Features

- Add New Books
- Purchase Books
- Search Books by Name
- Update Book Details
- Display All Books
- Sort Books Alphabetically
- Count Total Books
- Calculate Total Inventory Value
- Book Type Management (Story, Love, Educational)
- Menu-Driven User Interface

## Technologies Used
